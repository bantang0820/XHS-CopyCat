# XHS-CopyCat (小红书风格迁移助手)

这是一个基于 Streamlit 和 Gemini 多模态大模型的小红书风格迁移助手。它可以分析商品图片、对标账号主页和爆款笔记，自动生成账号装修方案和爆款文案。

## 功能特点

- **多模态分析**: 识别商品卖点、对标账号人设、爆款文案风格。
- **账号装修**: 自动生成符合对标风格的昵称和简介。
- **文案生成**: 自动生成包含标题、正文和标签的爆款笔记。
- **中间件 Prompt**: 提供可微调的 Prompt，方便用户二次创作。

## 安装与运行

1. **克隆项目或下载代码**

2. **安装依赖**
   建议使用 Python 3.10+ 环境。
   ```bash
   pip install -r requirements.txt
   ```

3. **配置 API Key**
   目前 API Key 已配置在 `api_client.py` 中。如需更换，请修改该文件中的 `API_KEY` 变量。

4. **运行应用**
   ```bash
   streamlit run app.py
   ```

## 使用流程

1. **区域 A**: 上传一张本次要推广的商品图片。
2. **区域 B**: 上传 3 张对标账号主页截图，以及 5 张爆款笔记截图。
3. **区域 C**: 点击“开始分析并生成”。
4. **区域 D**: 查看生成的账号装修方案、Prompt 和示例文案。

## 技术栈

- Python 3.10+
- Streamlit
- Google Gemini (via OpenRouter)
